HTML : HyperText Markup Language

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to descibe a web page’s appearance/presentation (CSS) or function/behavior (JavaScript)

« Hypertext » refers to link that connect web pages to one another, either within a single website or between websites. Links are fundamental aspect of the Web. By uploading content to the internet and linking it to pages created by other people, you became an active participant in the World Wide Web.

HTML uses « markup » to annotate texts, images and other content for display in a Web browser. HTML markup includes special « elements » such as <head>, <title>, <body>, <footer>, <article>, <section>, <p>, <div>, <span>, <img>, <aside>, <audio>, <canvas>, <datalist>, <embeded>, <nav>, <output>, <progress>, <video>, <ul>, <ol>, <li> and many others…

An HTML element is set off from other text in a document by « tags », which consist of the element name surrounded by « < » and « > ». The name of an element inside a tag is case sensitive. That is, it can be written in uppercase, lowercase or a mixture. For example, the < title > tag can be written as < Title >, < TITLE > or in any other way. However, the convention and recommended practice is to write tag in lowercase.
